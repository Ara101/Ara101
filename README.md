# 👋 Hi, I'm Lionel

*PhD student in Bioengineering, University of Washington · [Imoukhuede Lab](https://sites.bioe.uw.edu/imoukhuede/)*

Working on computational and quantitative modeling of oxytocin receptor (OXTR) signaling.

## 👀 What I'm interested in
- **Kinetic meta-analysis** of Class A GPCR signaling, using Bayesian hierarchical and random-effects modeling
- **Biased agonist design**, combining generative modeling (DrugEx), multi-objective reinforcement learning, QSPR modeling, and structure-based docking against the OXTR cryo-EM structure
- **Mechanistic ODE modeling** of receptor signaling, built from a literature-curated kinetic parameter database
- Staying active!

## 🌱 Currently
Learning deep learning, and preparing for my PhD general exam.

## 💞️ Looking to collaborate on
Computational, systems-level modeling of biological systems, especially GPCR pharmacology and signal transduction.
